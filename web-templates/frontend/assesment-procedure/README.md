# Assessment Procedure - Отчет по результатам оценки

Веб-шаблон для отображения отчета по результатам оценки.

## Структура проекта

- `src/api/api.ts` - API функции для работы с бекендом
- `src/pages/Report/Report.tsx` - Компонент отчета
- `src/app/App.tsx` - Точка входа приложения

## Установка и запуск

```bash
npm install
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5174`

## Сборка

```bash
npm run build
```

## Функциональность

Отчет отображает данные с группировкой:
1. По ФИО оцениваемого (сортировка по алфавиту)
2. По процедуре оценки (внутри каждой группы оцениваемого)

### Поля отчета:
- Процедура оценки
- Дата начала процедуры
- Дата завершения процедуры
- ФИО оцениваемого
- Текущий этап ДО плана оценки
- ФИО оценивающего
- Оценка

## Настройка бекенда

Бекенд должен быть настроен в `web-templates/backend/src/assesment-procedure/index.ts`
